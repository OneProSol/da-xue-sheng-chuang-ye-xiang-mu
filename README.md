### 一、相关技术
- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

学生功能：1. 登录、2. 注册、3. 首页、4. 公告消息、5. 校园资讯、6. 项目申报、7. 项目租队、8. 个人中心、9. 中期检查、10. 项目申报、11. 结题验收、12. 项目租队

老师功能：1. 登录、2. 中期检查、3. 项目申报

院级审批者功能：1. 登录、2. 中期检查、3. 项目申报、4. 结题验收

管理员功能：1. 登录、2. 轮播图管理、3. 公告栏管理、4. 用户管理、5. 校园资讯管理、6. 校园资讯分类管理、7. 中期检查管理、8. 项目申报管理、9. 结题验收管理、10. 项目租队管理、11. 项目共享管理
